Blog module

#javelin-blog

A core Javelin module for creating a blog. Aside from blog post styling, this component also generates styling for the pager, blog, rss link and tag cloud.

Display Order

By default, the LESS component for this module will generate styles that expect the blog post image to appear first in the display order, followed by the title, which is followed by either the date or summary.

Thumbnail Sizing

The size of the thumbnail generated through module settings that are editable by the user should be handled with CSS as well as module settings. To set this size globally, override the @blog-post-thumb-width var. To set this value on individual blog modules, you have to manually provide the CSS to handle the sizing.

comments Container for the comments section of the blog details view.
comments-form Container for the "add a comment" form in the comments section of the blog details view.
comments-item A single blog comment item in a list of blog comments.
comments-item-body Body block for a blog comment item.
comments-item-date Date block for a blog comment item.
comments-item-title Title block for a blog comment item.
comments-list List of comments in the comments section of the blog details view.
comments-title Title block for the comments section of the blog details view.
details-view Container for the blog details view.
index-view Container for the blog index view.
pager Container for the blog pager.
pager-item A single item in the blog pager.
pager-item-link A clickable item in the blog pager.
pager-next The "next" link in the blog pager.
pager-prev The "prev" link in the blog pager.
pager-prevnext The "prev" and "next" links in the blog pager.
post A single blog post item in a list of blog posts.
post-body Body block for a blog post item.
post-commentslink "Add a comment" link for a blog post item.
post-date Date block for a blog post item.
post-image Image block for a blog post item.
post-readmore "Read More" link for a blog post item.
post-summary Summary block for a blog post item.
post-title Title block for a blog post item.
rss RSS link for a blog post item.
tag Individual tags inside a tag block for a blog post item.
tags Tag block for a blog post item.
