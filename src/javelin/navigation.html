Navigation module

#javelin-nav

By far the most in-depth LESS component in the framework, the core Navigation module can be fully styled almost entirely by overriding the many variables that are setup to work with this component.

Nesting Limit

Variables target properties of items nested up to three levels deep. The third level's styling applies to all other descendants from that point.

navbar()

#javelin-nav.navbar(@dropdown; @flyout; @float)

Name Description Type
dropdown Optional If true, style dropdown menus. Defaults to true. boolean
flyout Optional If true, style flyout menus. Defaults to false. boolean
float Optional If true, float first-level navigation <li> items to the left. Defaults to true. boolean

stacked()

#javelin-nav.stacked(@flyout)

Name Description Type
flyout Optional If true, style flyout menus. Defaults to false. boolean

inline()

#javelin-nav.inline(@offset-horz; @color; @hover-color; @divider)

Name Description Type
offset-horz Left and right margins for list items. number
color Link text color. color
hover-color Link text color for hover states. color
divider Optional If provided, this will render a divider between each list item. color
