Gallery module

#javelin-gallery

The core Gallery module doesn't require much styling unless you are working with a unique usage of the module. Otherwise, you're mainly just styling the spacing of the module items.

Limiting items per row

It is common to enforce a specific number of items per row in a list of gallery items. This can be applied globally by setting a number value for the @gallery-column-count variable. Or this can be applied individually with the use of the #javelin-gallery.set-columns() mixin (see below).

Mixins for Legacy Selectors

Source: /less/javelin/gallery.less

Mixin Usage
caption The text block underneath the image block for a gallery item. This text DOES show up in the lightbox display.
description The text block underneath the caption block for a gallery item. This text does NOT show up in the lightbox display.
image Image block for a gallery item.
item A single gallery item in a list of gallery items.
list List container for the list of gallery items.
self Container for the gallery index view.

set-columns()

#javelin-gallery.set-columns(@count)

This mixin does not set sizing metrics for gallery items. Instead, it helps control the flow of gallery items with the use of clear and the :nth-child selector so items of varying heights stay within their respective rows.

Name Description Type
count Total number of gallery items per row. number

Usage

.gallery-example {
    #javelin-gallery.set-columns(4);
}


// Will output...
.gallery-example li.javelin_galleryItem:nth-child(4n+1) {
    clear: left;
}
.gallery-example li.javelin_galleryItem:nth-child(4n+4) {
    clear: right;
}
