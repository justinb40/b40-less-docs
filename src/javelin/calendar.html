Calendar module

#javelin-calendar

A core Javelin module for creating a calendar. Several LESS mixins for this module make it easier to style specific elements found within a calendar table including table cells, day numbers, weekends, days containing events and more.

Mixins for Legacy Selectors

Source: /less/javelin/calendar.less

Mixin Usage
backlink The "go back" link on the blog details view.
details-view Container for the calendar details view.
event A single event post item in a list of calendar events.
event-body Body block for a calendar event item.
event-date Date block for a calendar event item.
event-summary Summary block for a calendar event item.
event-title Title block for a calendar event item.
table Container for the calendar table.
table-cells All <th> and <td> cells in the calendar table.
table-day All <td> cells in the calendar table (which effectively ignores the top row of cells for the weekday labels).
table-day-event Event items found inside of a table cell.
table-day-hasevents Table cells that container event listings.
table-day-num The day number indicator inside of a table cell.
table-full Container for the full calendar table.
table-mini Container for the mini calendar table.
table-month Month block for the calendar table.
table-next The "next" link in the calendar table.
table-prev The "prev" link in the calendar table.
table-prevnext The "prev" and "next" links in the calendar table.
table-today The table cell that matches the current date.
table-weekday All <td> cells in the calendar table (which effectively ignores all cells except the top row).
table-weekend The far left and far right cells in a table.
