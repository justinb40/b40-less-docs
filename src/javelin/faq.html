FAQ module

#javelin-faq

The core FAQ module doesn't require much styling unless you are setting it up to behave as an accordion. Otherwise, you're mainly just styling text formatting or the spacing of the module items.

Mixins for Legacy Selectors

Source: /less/javelin/faq.less

Mixin Usage
answers-title Title block for the list of answers.
item A single answer item.
item-answer Answer block for an answer item.
item-question Question block for an answer item.
item-toplink The "back to top" link for an answer item.
questions-list Container for the list of questions.
questions-title Title block for the list of questions.
