Links module

#javelin-links

By default, the core Links module doesn't generate any default styles from the LESS framework and it also doesn't have any associated variables. However, it does have a short list of mixin selectors and can be used to easily style a list of links in an unlimited number of ways.

Mixins for Legacy Selectors

Source: /less/javelin/links.less

Mixin Usage
item A single link item in a list of links.
item-anchor Link block for a link item.
item-description Description block for a link item.
item-link This is just a <div> that wraps the actual link block... who knows.
list List container for the list of links items.
self Container for the links index view.

inline()

#javelin-gallery.inline(@offset-horz; @color; @hover-color; @divider)

The links module has its own mixin for putting items into a single line. Additionally, if the @divider argument is given, it will render dividers between each item.

Name Description Type
offset-horz Left and right margins. number
color Text color. color
hover-color Text color for hover states. color
divider Optional If provided, this will render a divider between each item. color

Usage

.links-example {
    #javelin-links.inline(10px; #08c; darken(#08c, 15%); #fff);
}
