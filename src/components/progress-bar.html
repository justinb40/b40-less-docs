Progress Bar component

Provide feedback on the progress of a goal or a process with simple yet flexible progress bars.

Sizing of Progress Bar Segments

Use a [style] attribute to control the size of the progress element.

Classes

Class Usage
progress-bar Container for a progress bar element.
progress A single segment within a progress-bar element, typically applied to a <div> element.

Usage

<div class="progress-bar">
    <div class="progress" style="width: 50%;"></div>
</div>


<!-- With % label -->
<div class="progress-bar">
    <div class="progress" style="width: 50%;">50%</div>
</div>


<!-- With % label that is only viewable to screen-readers -->
<div class="progress-bar">
    <div class="progress" style="width: 50%;">
        <span class="sr">50%</span>
    </div>
</div>
