Icon component

This is a placeholder class for managing icon elements. By default, no sprite image is associated with this component but that can be set with the @icon-sprite-path variable. Also be sure to properly set the values for the @icon-width and @icon-height variables.

Classes

Class Usage
icon Base class for icon elements. This is meant to be used with an icon- subclass to indicate the icon position within the sprite image.

Usage

.icon-facebook  { background-position: 0 0; }
.icon-twitter   { background-position: -30px 0; }
.icon-instagram { background-position: -60px 0; }
