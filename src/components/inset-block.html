Inset Block component

Create a neutral colored block with an inner shadow to give an "inset block" effect.

Classes

Class Usage
inset-block Wrap text in this class to create an inset block.

Usage

<div class="inset-block">
    <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.</p>
</div>
