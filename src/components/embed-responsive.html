Embed Responsive component

Automatically control the aspect ratio for embedded media, such as an embedded YouTube video.

Classes

Class Usage
embed-responsive Wrap embedded media with this class to enable responsive embed behavior.
embed-responsive-item An optional class to increase style specification for the embedded media. It is not typically necessary to use this class unless styling is present that is affecting the dimensions of the embedded media.
embed-responsive-16x9 Widescreen aspect ratio.
embed-responsive-4x3 Standard aspect ratio.

Usage

<div class="embed-responsive embed-responsive-16x9">
    <iframe src="http://youtube.com/embed/1234567890" allowTransparency="true" frameborder="0"></iframe>
</div>

<!-- If styling is present that is affecting the dimensions of the embedded media, using the 'embed-responsive-item' class will help work around that. -->
<div class="embed-responsive embed-responsive-16x9">
    <iframe src="http://youtube.com/embed/1234567890" allowTransparency="true" frameborder="0" class="embed-responsive-item"></iframe>
</div>
