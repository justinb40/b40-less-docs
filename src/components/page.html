Common Page Elements component

Automatically set z-index ordering for common page elements and provide standard styling for elements like the logo, page title and links inside the Back40 credit line.

Classes

Class Usage
b40-credit Wrap the javelin credit tag with this class to automatically style link colors to inherit to plain text color.
default-region Wrap a javelin region tag with this class to give the space around it a minimum height. This is useful when the space for the main content is empty and you do not want to the page to collapse entirely.
logo Because we typically place logos in an <h2>, this class will remove its bottom margin.
page Use as the outer-most container for your page structure to isolate your page from positioned elements like modals or third-party content injected into the DOM.
page-footer Wrap the footer of your page with a <footer> element using this class.
page-header Wrap the header of your page with a <header> element using this class.
page-hero If your site is using a hero image, wrap that section of your page with a <div> element using this class.
page-main Wrap the main content of your page with a <main> element using this class.
page-nav Wrap the main navigation of your page with a <nav> element using this class.
page-title If your page title is separate from the main content, wrap that section of your page with a <div> element using this class.

Usage

<body>
    <div class="page">
        <header class="page-header">
            <div class="container">
                
                <h2 class="logo">
                    <a href="/">
                        <img src="/sites/back40design/templates/default/images/logo.jpg" alt="Back40 Design">
                    </a>
                </h2>
                
                <nav class="page-nav">
                    <!nav id="nav" type="nested" /!>
                </nav>
                
            </div>
        </header>
        <div class="page-hero">
            <div class="container">
                
                <!region:hero type="global" /!>
                
            </div>
        </div>
        <main class="page-main">
            <div class="container">
                
                <div class="default-region">
                    <!region:default type="default" /!>
                </div>
                
            </div>
        </main>
        <footer>
            <div class="container">
                
                &copy;2015 Back40 Design
                <span class="b40-credit"> <!back40:credit /!></span>
                
            </div>
        </footer>
    </div>
</body>
