Horizontal Form component

Use grid classes to align labels and groups of form controls in a horizontal layout. Doing so changes form-group elements to behave as grid rows, so there is no need to use the row class in this context.

When @is-responsive is set to true, styling only applies to labels within viewports that are at least 768px.

Classes

Class Usage
form-horizontal Apply to either a <form>, <fieldset> or <div> to enable properties for horizontal form layout.

Usage

<form action="#" method="post" class="form-horizontal">
    <div class="form-group">
        <label class="grid-3">First Name</label>
        <div class="grid-9">
            <input type="text" class="form-control" placeholder="First Name">
        </div>
    </div>
    <div class="form-group">
        <label class="grid-3">Last Name</label>
        <div class="grid-9">
            <input type="text" class="form-control" placeholder="Last Name">
        </div>
    </div>
    <div class="form-group">
        <label class="grid-3">Email</label>
        <div class="grid-9">
            <input type="text" class="form-control" placeholder="Email">
        </div>
    </div>
</form>
