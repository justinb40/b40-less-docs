Simple Modal component

Use the Simple Modal jQuery plugin to easily display content within a modal overlay. This component will set some basic styles for the modal container and the modal close button.

When displaying an embedded video within the modal, you can use the data-simplemodal="open-simplemodal-video" attribute on a <a> element to automatically configure Simple Modal to embed and display the video from the href attribute. See below for implementation details.

Official Simple Modal Documentation

JavaScript Dependency

This component relies on additional logic to handle the Simple Modal plugin. Include these scripts at the bottom of your document before the closing </body> tag.


<script src="public/scripts/simplemodal.min.js"></script>
<script src="public/scripts/less/less.common.min.js"></script>

Alertnative JavaScript Dependency

This component relies on additional logic to handle loading videos in the modal. Include this script along with the ones listed above.


<script src="public/scripts/less/less.simplemodal-video.min.js"></script>

Classes

Class Usage
simplemodal-close Close button for the modal.
simplemodal-container Main container for the modal.
simplemodal-data Container for the modal content.
simplemodal-overlay Container for the background overlay.
simplemodal-wrap Additional wrapper within the modal-container. Honestly not sure what its real purpose is.

JS API

Attribute Usage
data-simplemodal Set the value of this attribute to open-simplemodal-video on a clickable element to open its href attribute into the modal as an embedded video.
data-width Set the width for the video that will be loaded into the modal. Defaults to 800.
data-height Set the height for the video that will be loaded into the modal. Defaults to 450.
data-autoplay Set the flag for whether or not to immediately play the video once the modal is open. Defaults to 0.

Usage

<a href="//player.vimeo.com/video/142512345" class="btn btn-default" data-simplemodal="open-simplemodal-video" data-autoplay="1">Open Video</a>

JavaScript Events

Event Usage
b40.simplemodal.open Broadcast when the modal is opened and runs the onOpen() callback.
b40.simplemodal.close Broadcast when the modal is closed and runs the onClose() callback.
b40.simplemodal.open Broadcast when the modal is shown and runs the onShow() callback.

Using Events

// Bind to the `document` to listen for Simple Modal events
$('body').bind('b40.simplemodal.open', function(dialog) {
    // `dialog` is passed from the Simple Modal `onOpen` callback (same with other events)
});
