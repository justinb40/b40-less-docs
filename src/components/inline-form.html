Inline Form component

Wrap form content with .form-inline for left-aligned and inline-block form controls.

The form-control class has width: 100%; applied by default. Within inline forms, we reset that to width: auto; so multiple controls can reside on the same line. Depending on your layout, specifying an inline [style] attribute for a custom width may be required.

When @is-responsive is set to true, this only applies to forms within viewports that are at least 768px wide.

Screen readers will have trouble with your forms if you don't include a <label> with every <input>.

Classes

Class Usage
form-inline Apply to either a <form>, <fieldset> or <div> to enable properties for inline form layout.

Usage

<form action="#" method="post" class="form-inline">
    <div class="form-group">
        <label class="sr">First Name</label>
        <input type="text" class="form-control" placeholder="Username">
    </div>
    <div class="form-group">
        <label class="sr">Last Name</label>
        <input type="password" class="form-control" placeholder="Password">
    </div>
    <button type="button" class="btn btn-default">Login</button>
</form>
