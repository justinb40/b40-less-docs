Search Bar component

JavaScript Dependency

This component relies on additional logic to handle toggling the collapsed state of the search bar. Include these scripts at the bottom of your document before the closing </body> tag.


<script src="public/scripts/less/less.common.min.js"></script>
<script src="public/scripts/less/less.search-collapse.min.js"></script>

Classes

Class Usage
search-bar Container for the search bar.
search-bar-input The text input element in the search bar.
search-bar-btn The button element in the search bar.
search-bar-collapsible An additional class to attach to the search-bar class.

Usage

<div class="search-bar search-bar-collapsible">
    <form action="/search/" method="get">
        <input type="text" name="query" class="search-bar-input" value="" placeholder="Search...">
        <button class="search-bar-btn" type="submit">
            <span class="icon icon-search"></span>
            <span class="sr">Search</span>
        </button>
    </form>
</div>

JavaScript Events

Event Usage
b40.search.expanded Broadcast when the search bar is expanded.
b40.search.collapsed Broadcast when the search bar is collapsed.

Using Events

// Bind to the `document` to listen for search bar events
$('body').bind('b40.search.expanded', function() {
    // code goes here
});
