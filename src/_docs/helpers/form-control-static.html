form-control-static helper

Set text blocks to have the same vertical box-model properties as a .form-control element.

Usage

<div class="form-group">
    <label>Form Label</label>
    <div class="form-inline">
        <input type="text" class="form-control text-right" placeholder="http://subdomain">
        <span class="form-control-static">.domain.com</span>
    </div>
</div>
