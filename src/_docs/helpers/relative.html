relative helper

Apply a relative position to an element. This is helpful when you need to position child elements but have no strong justification for creating a new parent class to handle it.

Usage

<div class="row relative">
    <div class="grid-6">
        Column 1
    </div>
    <hr class="hr-vertical">
    <div class="grid-6">
        Column 2
    </div>
</div>
