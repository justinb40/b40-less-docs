Grid component

Grid systems are used for creating page layouts through a series of rows and columns that contain your content.

Features:

  • Standard layout or responsive
  • 12-column grid
  • Fixed-width container
  • Fluid layout columns
  • Infinite nesting

Responsive Grid

Setting the @is-responsive variable to true will generate a completely different set of grid class names to handle various breakpoints for handling layout on different device sizes.

Classes

Class Usage
container Use on a <div> to create a centered, fixed-width (960px) column on the page.
container-fluid Use on a <div> to create a centered, fluid-width (100%) column on the page.
row Use on a <div> nested within a container to create a grid row that will properly clear grid columns. With the exception of the vertical <hr> component, their only immediate children should be grid columns.
These classes are only generated when @is-responsive is set to false.
grid-[n] Create grid columns inside a row.
push-[n] Reorder grid columns by moving them to the right.
pull-[n] Reorder grid columns by moving them to the left.
offset-[n] Move columns to the right by using offset classes. This also works very well with the vertical <hr> component.
These classes are only generated when @is-responsive is set to true.
col-[size]-[n] Create responsive grid columns inside a row.
push-[size]-[n] Reorder grid columns by moving them to the right with classes corresponding to different device-width breakpoints on responsive sites.
pull-[size]-[n] Reorder grid columns by moving them to the left with classes corresponding to different device-width breakpoints on responsive sites.
offset-[size]-[n] Move columns to the right by using offset classes corresponding to different device-width breakpoints on responsive sites. This also works very well with the vertical <hr> component.

Usage

<div class="container">
    <div class="row">
        <div class="grid-4">Column 1</div>
        <div class="grid-4">Column 2</div>
        <div class="grid-4">Column 3</div>
    </div>
</div>


<!-- Offsetting column position -->
<div class="container">
    <div class="row">
        <div class="grid-6 offset-3">Centered column with offsets</div>
    </div>
</div>


<!-- Controlling column display order -->
<div class="container">
    <div class="row">
        <div class="grid-6 push-6">This column will display second on the row</div>
        <div class="grid-6 pull-6">This column will display first on the row</div>
    </div>
</div>


<!-- Configuring responsive column display -->
<!-- col-sm-12 will render the column across the entire row for device widths between 480px and 768px -->
<!-- col-md-6 will render the column across half the row for device widths between 768px and 990px -->
<div class="container">
    <div class="row">
        <div class="col-sm-12 col-md-6">Grid Column</div>
        <div class="col-sm-12 col-md-6">Grid Column</div>
    </div>
</div>
