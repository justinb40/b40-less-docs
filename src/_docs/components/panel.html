Panel component

While not always necessary, sometimes you need to put your DOM in a box that can handle headers, footers and other content containers. For those situations, try the panel component.

Classes

Class Usage
panel Container class typically applied to a <div>.
panel-body Container for normal panel content. Tables may be placed outside of this element as an immediate child of the panel element.
panel-footer Container for the panel footer.
panel-header Container for the panel header.
panel-media Container for main panel images or video embedded through an <iframe>.
panel-title Apply to a heading tag (probably an <h4>) within a panel-header element.

Usage

<div class="panel">
    <div class="panel-header">
        <h4 class="panel-title">Panel Title</h4>
    </div>
    <div class="panel-media">
        <img src="http://dummyimage.com/1600x800/333/fff&text=+">
    </div>
    <div class="panel-body">
        <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.</p>
    </div>
    <div class="panel-footer">
        <a href="#" class="btn btn-default btn-block">Learn More</a>
    </div>
</div>
