Form Blocks component

Gain better control over spacing and organization of form elements.

Classes

Class Usage
form-section Use on a <div> or <fieldset> to add spacing between consecutive form-section elements.
form-group Use on a <div> to help with the organization and spacing on vertical forms.

Usage

<fieldset class="form-section">
    <legend>Your Name</legend>
    
    <div class="form-group">
        <label>First Name</label>
        <input type="text" name="firstname" class="form-control">
    </div>
    <div class="form-group">
        <label>Last Name</label>
        <input type="text" name="lastname" class="form-control">
    </div>
</fieldset>

<fieldset class="form-section">
    <legend>Contact Info</legend>
    
    <div class="form-group">
        <label>Phone</label>
        <input type="text" name="phone" class="form-control">
    </div>
    <div class="form-group">
        <label>Email</label>
        <input type="text" name="email" class="form-control">
    </div>
</fieldset>
