text-overflow mixin

.text-overflow()

Truncate the text in a block element and append an ellipsis (...) at the end.

Parameters

None.

Usage

.example {
    .text-overflow();
    
    // Only display as much text that will within 300px
    width: 300px;
}


// Will output...
.example {
    overflow: hidden;
    text-overflow: ellipsis;
    white-space: nowrap;
    
    width: 300px;
}
