button-color mixin

.button-color(@bg; @color; @border; @hover-bg; @hover-color; @hover-border)

Sets color properties for static, :hover, :focus, :active and disabled options for buttons.

Optional Arguments

If hover state colors are omitted in the arguments list, those values are still set as darkened versions of the first set of colors.

Parameters

Name Description Type
bg Background color. color
color Text color. color
border Border color. color
hover-bg Optional Background color for hover states. color
hover-color Optional Text color for hover states. color
hover-border Optional Border color for hover states. color

Usage

.btn-example {
    .button-color(#08c; #fff; #08c; darken(#08c, 15%); darken(#fff, 15%); darken(#08c, 15%));
}


// Will output...
.btn-example {
    background-color: #08c;
    border-color: #08c;
    color: #fff;
}

.btn-example:hover,
.btn-example:focus,
.btn-example:active,
.btn-example.active {
    background-color: #005580;
    border-color: #005580;
    color: #d9d9d9;
}

.btn-example.disabled,
.btn-example.disabled:hover,
.btn-example.disabled:focus,
.btn-example.disabled:active,
.btn-example.disabled.active,
.btn-example[disabled],
.btn-example[disabled]:hover,
.btn-example[disabled]:focus,
.btn-example[disabled]:active,
.btn-example[disabled].active,
fieldset[disabled] .btn-example,
fieldset[disabled] .btn-example:hover,
fieldset[disabled] .btn-example:focus,
fieldset[disabled] .btn-example:active,
fieldset[disabled] .btn-example.active {
    background-color: #005580;
    border-color: #005580;
}
