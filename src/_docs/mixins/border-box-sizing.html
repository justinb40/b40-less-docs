border-box-sizing mixin by reference

.border-box-sizing (reference)

Allows an element to modify sizing metrics without altering the explicit width or height.

Reference Only

This class is meant to be extended rather than called as a mixin.

Parameters

None.

Usage

.block {
    &:extend(.border-box-sizing all);
    
    width: 100%;
    padding: 30px;
}


// Will output...
.block {
    width: 100%;
    padding: 30px;
    
    -webkit-box-sizing: border-box;
       -moz-box-sizing: border-box;
            box-sizing: border-box;
}
