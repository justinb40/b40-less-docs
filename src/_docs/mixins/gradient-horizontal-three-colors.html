gradient.horizontal-three-colors mixin

#gradient.horizontal-three-colors(@start-color; @mid-color; @end-color; @mid-percent)

Apply a horizontal, three-color gradient to the background-image property of an element.

Parameters

Name Description Type
start-color Left color (start) for the gradient image. color
mid-color Middle color for the gradient image. color
end-color Right color (end) for the gradient image. color
mid-percent Optional Color-stop position for the middle color. percent

Usage

.gradient-example {
    #gradient.horizontal-three-colors(#08c; darken(#08c, 15%); darken(#08c, 30%));
}


// Will output...
.gradient-example {
    background-image: -webkit-linear-gradient(left, #0088cc, #005580 50%, #002233);
    background-image: -o-linear-gradient(left, #0088cc, #005580 50%, #002233);
    background-image: linear-gradient(to right, #0088cc, #005580 50%, #002233);
    background-repeat: repeat-y;
    filter: progid:DXImageTransform.Microsoft.gradient(startColorstr='#ff0088cc', endColorstr='#ff002233', GradientType=1);
}
