overflow-scroll mixin

.overflow-scroll(@direction)

Set overflow property of an element (typically a <div>) .

If direction is omitted, it applies the overflow to both directions.

Parameters

Name Description Type
direction Optional Which direction to apply the overflow scrolling, either x or y. keyword

Usage

.example-scrollable-div {
    .overflow-scroll(y);
    
    width: 100%;
    height: 400px;
}


// Will output...
.example-scrollable-div {
    width: 100%;
    height: 400px;
    overflow-y: auto;
    
    -webkit-overflow-scrolling: touch;
}
