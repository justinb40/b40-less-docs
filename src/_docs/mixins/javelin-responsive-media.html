javelin-responsive-media mixin

.javelin-responsive-media(@selector)

When dealing with responsive media like images or iframes, this function can be used to make those objects responsive.

Selector stacks will not work with the @selector argument. Only single selectors will correctly accept the styling from the mixin.

Parameters

Name Description Type
selector A single class name selector (without the decimal) to apply the mixin rules. keyword

Usage

.javelin-responsive-media(javelin_textItem);


// Will output...
.javelin_textItem {
    img,
    iframe {
        -webkit-box-sizing: border-box;
           -moz-box-sizing: border-box;
                box-sizing: border-box;
    }
    
    img {
        max-width: 100%;
        height: auto !important;
        
        -webkit-transform: translate3d(0, 0, 0);
                transform: translate3d(0, 0, 0);
    }
    
    iframe {
        max-width: 100%;
    }
}


// This type of thing...
.javelin-responsive-media(~".javelin_textItem, .javelin_galleryItem, .javelin_faqItem");

// Fails in LESS...
// (an additional decimal is created and the img and iframe selectors do not apply to every selector in the stack)
..javelin_textItem,
.javelin_galleryItem,
.javelin_faqItem img,
..javelin_textItem,
.javelin_galleryItem,
.javelin_faqItem iframe {
    -webkit-box-sizing: border-box;
       -moz-box-sizing: border-box;
            box-sizing: border-box;
}

..javelin_textItem,
.javelin_galleryItem,
.javelin_faqItem img {
    max-width: 100%;
    height: auto !important;
    
    -webkit-transform: translate3d(0, 0, 0);
            transform: translate3d(0, 0, 0);
}

..javelin_textItem,
.javelin_galleryItem,
.javelin_faqItem iframe {
    max-width: 100%;
}
