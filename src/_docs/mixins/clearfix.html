clearfix mixin by reference

.clearfix (reference)

Allows a parent container to clear child floats with no additional markup.

Reference Only

This class is meant to be extended rather than called as a mixin.

Parameters

None.

Usage

.parent {
    &:extend(.clearfix all);
    
    .child {
        width: 25%;
        float: left;
    }
}


// Will output...
.parent:before,
.parent:after {
    content: "";
    display: table;
}
.parent:after {
    clear: both;
}
.parent .child {
    width: 25%;
    float: left;
}
