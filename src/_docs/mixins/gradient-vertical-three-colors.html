gradient.vertical-three-colors mixin

#gradient.vertical-three-colors(@start-color; @mid-color; @end-color; @mid-percent)

Apply a vertical, three-color gradient to the background-image property of an element.

Parameters

Name Description Type
start-color Top color (start) for the gradient image. color
mid-color Middle color for the gradient image. color
end-color Bottom color (end) for the gradient image. color
mid-percent Optional Color-stop position for the middle color. percent

Usage

.gradient-example {
    #gradient.vertical-three-colors(#08c; darken(#08c, 15%); darken(#08c, 30%));
}


// Will output...
.gradient-example {
    background-image: -webkit-linear-gradient(top, #0088cc, #005580 50%, #002233);
    background-image: -o-linear-gradient(top, #0088cc, #005580 50%, #002233);
    background-image: linear-gradient(to bottom, #0088cc, #005580 50%, #002233);
    background-repeat: repeat-x;
    filter: progid:DXImageTransform.Microsoft.gradient(startColorstr='#ff0088cc', endColorstr='#ff002233', GradientType=0);
}
