defer-typekit-styles mixin

.defer-typekit-styles(@beforeRules; @afterRules)

Some custom fonts may look drastically different in size while Typekit is initializing itself and loading your custom fonts. To prevent the flash of unstyled content (FOUC), use this mixin to provide a before ruleset to help maintain spacing and layout and an after ruleset to apply the final styling that uses properties for the custom font.

Parameters

Name Description Type
before-rules The styling rules to apply before Typekit has fully loaded custom fonts. ruleset
after-rules The styling rules to apply after Typekit has fully loaded custom fonts. ruleset

Usage

h1 {
    .defer-typekit-styles({
        font-size: floor(@h1-size / 2);
    }; {
        font-size: @h1-size;
    });
}


// Will output...
h1 {
    font-size: 16px;
}

.wf-active h1 {
    font-size: 32px;
}
