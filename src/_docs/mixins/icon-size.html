icon-size mixin

.icon-size(@width; @height)

Set width and height for an icon.

If height is omitted, it assumes the value of width.

Parameters

Name Description Type
width Icon width. number
height Optional Icon height. number

Usage

.example-icon {
    .icon-size(40px);
}


// Will output...
.example-icon {
    width: 40px;
    height: 40px;
}
