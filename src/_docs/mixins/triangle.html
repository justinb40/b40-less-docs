triangle mixin

.triangle(@side; @width; @height; @color)

Generate pure CSS triangles.

CSS triangles can never be obtuse.

Parameters

Name Description Type
direction Which direction the triangle vertex (top) will point. keyword
width Width. number
height Height. number
color Solid color for triangle. color

Usage

.caret::after {
    .triangle(top; 20px; 10px; #08c);
    
    margin-left: -10px;
    position: absolute;
    top: -10px;
    left: 50%;
    content: "";
}


// Will output...
.caret::after {
    // This positions the triangle
    margin-left: -10px;
    position: absolute;
    top: -10px;
    left: 50%;
    content: "";
    
    // This generates the triangle itself, with the width (20px) divided evenly to the left and right borders
    border-bottom: 10px solid #08c;
    border-left: 10px solid transparent;
    border-right: 10px solid transparent;
}
