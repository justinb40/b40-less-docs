make-grid-container mixin

.make-grid-container(@gutter)

Apply grid container properties to create a centered container on the page.

Parameters

Name Description Type
gutter Optional Gutter width. number

Usage

.example-container {
    .make-grid-container(30px);
}


// Will output...
.example-container::before,
.example-container::after {
    content: "";
    display: table;
}
.example-container::after {
    clear: both;
}
.example-container {
    width: 990px;
    margin-left: auto;
    margin-right: auto;
    padding-left: 15px;
    padding-right: 15px;
    
    -webkit-box-sizing: border-box;
       -moz-box-sizing: border-box;
            box-sizing: border-box;
}
