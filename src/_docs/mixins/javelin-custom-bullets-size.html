javelin-custom-bullets-size mixin

.javelin-custom-bullets(@bg-size)

Set a custom background-size property for custom list bullets in Javelin modules when dealing with retina display.

Retina Display

This mixin still requires you to wrap your own call to the mixin with a media query for targeting retina display.

Parameters

Name Description Type
bg-size The background image size. width height

Usage

@media (min-resolution: 192dpi), (-webkit-min-device-pixel-ratio: 2), (min--moz-device-pixel-ratio: 2), (-o-min-device-pixel-ratio: 2/1), (min-device-pixel-ratio: 2), (min-resolution: 2dppx) {
    .javelin-custom-bullets-size(96px 16px);
}


// Will output...
@media (min-resolution: 192dpi), (-webkit-min-device-pixel-ratio: 2), (min--moz-device-pixel-ratio: 2), (-o-min-device-pixel-ratio: 2/1), (min-device-pixel-ratio: 2), (min-resolution: 2dppx) {
    .javelin_textItem ul li,
    .javelin_rotatorItem ul li,
    .javelin_calendarEventBody ul li,
    .javelin_blogPostDescription ul li,
    .javelin_faqQuestions ul li,
    .javelin_faqItemAnswer ul li,
    .javelin_links ul li,
    .javelin_staffBody ul li,
    .cms-tabs-panel ul li {
        background-size: 96px 16px;
    }
}
