Twitter module

#javelin-twitter

The core Twitter module connects with the Twitter API to display recent tweets on your web page.

Mixins for Legacy Selectors

Source: /less/javelin/twitter.less

Mixin Usage
self Container for the twitter index view.
header Header block for a twitter feed.
footer Fooer block for a twitter feed.
usericon User icon block for a twitter feed.
userimage User image block for a twitter feed.
screenname Screenname block for a twitter feed.
username Username block for a twitter feed.
tweets Container list for a list of tweets.
tweet A single tweet item in a list of tweets.
tweet-text Text block for a tweet item.
tweet-meta Meta block for a tweet item.
