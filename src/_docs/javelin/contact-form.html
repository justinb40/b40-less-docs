Contact Form module

#javelin-contact

The core contact module in Javelin does not have many classified elements like other modules do. Instead, we have to style to the structure of the markup in the module. Furthermore, the contact module uses ID's on its container and input elements, which limits our usage of the module to just one per page and goes against the paradigm of using classes for styling as opposed to IDs.

Container Width

The container for this module will take on a width of 50% by default to handle using this module inside of a wide page layout. This can be changed by overriding the @contact-form-width var.

Input Field Sizes

A default width of 100% is applied to input fields by using the form-control helper as a mixin. Some fields take on a shorter width to better suit shorter lengths of expected input.
