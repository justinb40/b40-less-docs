reset-grid-container mixin

.reset-grid-container()

Reset grid container properties generated by the make-grid-container() mixin.

This mixin does not undo styles generated by clearfix() or border-box-sizing() mixins.

Parameters

None.

Usage

.example-container {
    .reset-grid-container();
}


// Will output...
.example-container {
    margin-left: 0;
    margin-right: 0;
    padding-left: 0;
    padding-right: 0;
}
