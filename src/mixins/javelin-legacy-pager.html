javelin-legacy-pager mixin

.javelin-legacy-pager(@size; @item-offset @default-bg; @default-color; @hover-bg; @hover-color; @active-bg; @active-color; @border-radius: 0;)

User this mixin to easily style the legacy pager that comes from Javelin.

Parameters

Name Description Type
size A square size to apply to each pager item (excluding previous and next links). number
item-offset The left and right margins applied to each pager item. number
default-bg Default background color for pager items. color
default-color Default text color for pager items. color
hover-bg Background color for hover states. color
hover-color Text color for hover states. color
active-bg Background color for active states (aka "current page"). color
active-color Text color for active states (aka "current page"). color
border-radius Optional Border radius size. number

 

Mixins for Legacy Selectors

Source: /less/mixins/javelin.less

All mixins take a single ruleset as an argument: { }
Mixin Usage
javelin-legacy-pager-item A single item in the pager, excluding the previous and next links.
javelin-legacy-pager-item-active An active or "current page" item.
javelin-legacy-pager-item-inactive Any item that is not an active or "current page" item.
javelin-legacy-pager-next The next page link.
javelin-legacy-pager-prev The previous page link.
javelin-legacy-pager-prevnext Both the previous and next links.

Usage

.cms-directory-pager {
    .javelin-legacy-pager(30px; 1px; #fff; #000; #fff; @brand-primary; @brand-primary; #fff; 50%);
    
    // Previous and Next links
    .javelin-legacy-pager-prevnext({
        .transition(opacity @animation-duration-base @animation-timing-function-base);
        
        opacity: 1;
        
        &:hover {
            opacity: .5;
        }
    });
    
    .javelin-legacy-pager-prev({
        float: left;
    });
    
    .javelin-legacy-pager-next({
        float: right;
    });
}
