form-control-focus mixin

.form-control-focus(@color)

Set border color and focus ring color for the :focus state of form controls.

Parameters

Name Description Type
color Color for the border and focus ring color. color

Usage

.form-control-example {
    .form-control-focus(#08c);
}


// Will output...
.form-control-example:focus {
    border-color: #08c;
    outline: 0;
    box-shadow: inset 0 1px 1px rgba(0, 0, 0, .075), 0 0 8px rgba(0, 136, 204, .6);
}
