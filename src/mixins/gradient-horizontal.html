gradient.horizontal mixin

#gradient.horizontal(@start-color; @end-color; @start-percent; @end-percent)

Apply a horizontal, two-color gradient to the background-image property of an element.

Parameters

Name Description Type
start-color Left color (start) for the gradient image. color
end-color Right color (end) for the gradient image. color
start-percent Optional Color-stop position for the start color. percent
end-percent Optional Color-stop position for the end color. percent

Usage

.gradient-example {
    #gradient.horizontal(#08c; darken(#08c, 15%));
}


// Will output...
.gradient-example {
    background-image: -webkit-linear-gradient(left, #08c 0%, #005580 100%);
    background-image: -o-linear-gradient(left, #08c 0%, #005580 100%);
    background-image: linear-gradient(to right, #08c 0%, #005580 100%);
    background-repeat: repeat-y;
    filter: progid:DXImageTransform.Microsoft.gradient(startColorstr='#ff0088cc', endColorstr='#ff005580', GradientType=1); 
}
