text-hide mixin

.text-hide()

Neatly hide the text within an element.

Avoid text-indent

Using text-indent: -9999px causes an issue with "tab focus" which will visibly render the focus outline across the page.

Parameters

None.

Usage

.logo {
    .text-hide();
    
    width: 300px;
    height: 100px;
    background-image: url(logo.png);
}


// Will output...
.logo {
    background-color: transparent;
    border: 0;
    color: transparent;
    font: 0/0 a;
    text-shadow: none;
    
    width: 300px;
    height: 100px;
    background-image: url(logo.png);
}
