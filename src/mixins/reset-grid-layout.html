reset-grid-layout mixin

.reset-grid-layout()

Reset grid row and column properties generated by the make-grid-row() and make-grid-column-default() mixins.

Since the @is-responsive flag determines the class name of the grid columns (either grid or col), this mixin will take that into consideration when resetting the grid styles.

This mixin does not undo styles generated by clearfix() or border-box-sizing() mixins.

Parameters

None.

Usage

.example {
    .reset-grid-layout();
}


// Will output...
.example .row {
    margin-left: 0;
    margin-right: 0;
}

.example [class*="grid-"] { // or [class*="col-"] when @is-responsive is `true`
    width: auto;
    min-height: 0;
    padding-left: 0;
    padding-right: 0;
    position: static;
    float: none;
}
