gradient.radial mixin

#gradient.radial(@inner-color; @outer-color; @shape)

Apply a radial gradient to the background-image property of an element.

Parameters

Name Description Type
inner-color Inner color (start) for the gradient image. color
outer-color Outer color (end) for the gradient image. color
shape Optional Shape of the gradient image. keyword

Usage

.gradient-example {
    #gradient.radial(darken(#08c, 15%); #08c);
}

.gradient-example {
    #gradient.radial(darken(#08c, 15%); #08c; ellipse);
}


// Will output...
.gradient-example {
    background-image: -webkit-radial-gradient(circle, #005580, #08c);
    background-image: radial-gradient(circle, #005580, #08c);
    background-repeat: no-repeat; 
}

.gradient-example {
    background-image: -webkit-radial-gradient(ellipse, #005580, #08c);
    background-image: radial-gradient(ellipse, #005580, #08c);
    background-repeat: no-repeat; 
}
