reset-grid-column mixin

.reset-grid-column()

Reset grid column properties generated by the make-grid-column-default() mixin. Since we also typically include a width and float along with this mixin, we reset those properties as well.

This mixin does not undo styles generated by clearfix() or border-box-sizing() mixins.

Parameters

None.

Usage

.example-column {
    .reset-grid-column();
}


// Will output...
.example-column {
    width: auto;
    min-height: 0;
    padding-left: 0;
    padding-right: 0;
    position: static;
    float: none;
}
