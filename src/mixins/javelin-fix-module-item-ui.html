javelin-fix-module-item-ui mixin

.javelin-fix-module-item-ui(@module)

By default, Javelin relies on static positioning to set a position on certain admin interface elements which can sometimes prevent the module actions from properly positioning near its module item.

This mixin will help correct this issue by applying a top and left positions equal to 0 to the module actions.

Parameters

Name Description Type
module The module shortname to use in the element selector. keyword

Usage

.javelin-fix-module-item-ui(directory);


// Will output...
.javelin_module-directory .javelin_moduleContent {
    top: 0 !important;
    left: 0 !important;
}
