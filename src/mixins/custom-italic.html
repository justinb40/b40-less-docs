custom-italic mixin

.custom-italic(@italic-family)

Set correct font styles and font families for custom fonts inside <i> and <em> tags.

Parameters

Name Description Type
italic-family Font family to use for italic formatting. string

Usage

.custom-italic(@font-family-proximanova-italic);


// Will output...
i,
em {
    font-family: ProximaNova-Italic, Arial, Helvetica, sans-serif;
    font-weight: normal;
}
