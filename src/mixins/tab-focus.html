tab-focus mixin

.tab-focus()

This accessibility feature will set the outline property to indicate when an element has focus.

You must call this mixin within the :focus state selector.

Parameters

None.

Usage

a:focus {
    .tab-focus();
}


// Will output...
a:focus {
    // Default
    outline: thin dotted;
     
    // WebKit
    outline: 5px auto -webkit-focus-ring-color;
    outline-offset: -2px;
}
