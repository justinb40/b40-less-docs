reset-grid-row mixin

.reset-grid-row()

Reset grid row properties generated by the make-grid-row() mixin.

This mixin does not undo styles generated by clearfix() or border-box-sizing() mixins.

Parameters

None.

Usage

.example-row {
    .reset-grid-row();
}


// Will output...
.example-row {
    margin-left: 0;
    margin-right: 0;
}
