form-control-size mixin

.form-control-size(@input-height; @padding-vert; @padding-horz; @font-size; @line-height; @border-radius)

Set sizing metrics for a form control.

This does not apply to checkboxes, radio buttons or file inputs.

Parameters

Name Description Type
input-height Total height of the input field. number
padding-vert Top and bottom padding. number
padding-horz Left and right padding. number
font-size Font size. number
line-height Line height. number
border-radius Optional Border radius size. number

Usage

.form-control-example {
    .form-control-size(30px; 5px; 10px; @font-size-sm; @line-height-sm; @border-radius-sm);
}


// Will output...
.form-control-example {
    height: 30px;
    padding: 5px 10px;
    border-radius: 3px;
    font-size: 12px;
    line-height: 1.5;
}
