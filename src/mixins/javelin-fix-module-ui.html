javelin-fix-module-ui mixin

.javelin-fix-module-ui(@module)

By default, Javelin does not set a position on certain admin interface elements which can sometimes prevent the module settings dropdown from staying on top of the module content.

This mixin will help correct this issue by applying a relative position to the module content. However, this causes issues with the modal dialog box because the same selector is used there. Luckily, this mixin fixes that problem as well.

Parameters

Name Description Type
module The module shortname to use in the element selector. keyword

Usage

.javelin-fix-module-ui(carousel);


// Will output...
.javelin_module-carousel .javelin_moduleContent {
    position: relative;
}
.javelin_dialog .javelin_module-carousel .javelin_moduleContent {
    position: static;
}
