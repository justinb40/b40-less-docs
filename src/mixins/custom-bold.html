custom-bold mixin

.custom-bold(@bold-family)

Set correct font styles and font families for custom fonts inside <b> and <strong> tags.

Parameters

Name Description Type
bold-family Font family to use for bold formatting. string

Usage

.custom-bold(@font-family-proximanova-bold);


// Will output...
b,
strong {
    font-family: ProximaNova-Bold, Arial, Helvetica, sans-serif;
    font-weight: normal;
}
