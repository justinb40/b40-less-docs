alert-color mixin

.alert-color(@bg; @border; @color)

Set color properties of an alert box. Additionally, colors for <hr> tags and the .alert-link class are automatically darkened using the colors provided.

Parameters

Name Description Type
bg Background color. color
border Border color. color
color Text color. color

Usage

.alert-example {
    .alert-color(#08c; #08c; #fff);
}


// Will output...
.alert-example {
    background-color: #08c;
    border-color: #08c;
    color: #fff;
}
.alert-example hr {
    border-color: #069;
}
.alert-example .alert-link {
    color: #f2f2f2;
}
