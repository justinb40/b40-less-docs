gradient.striped mixin

#gradient.striped(@color; @angle)

Apply a striped gradient to the background-image property of an element.

Parameters

Name Description Type
color The color of the stripe to be repeated. color
angle The angle of the direction, specified in degrees. number
start-percent Color-stop position for the start color. percent
mid-percent Color-stop position for the mid color. percent
end-percent Color-stop position for the end color. percent

Usage

.gradient-example {
    #gradient.striped(#08c);
}


// Will output...
.gradient-example {
    background-image: -webkit-linear-gradient(45deg, #08c 25%, transparent 25%, transparent 50%, #08c 50%, #08c 75%, transparent 75%, transparent);
    background-image: -o-linear-gradient(45deg, #08c 25%, transparent 25%, transparent 50%, #08c 50%, #08c 75%, transparent 75%, transparent);
    background-image: linear-gradient(45deg, #08c 25%, transparent 25%, transparent 50%, #08c 50%, #08c 75%, transparent 75%, transparent);
}
