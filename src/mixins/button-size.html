button-size mixin

.button-size(@height; @font-size; @line-height; @padding-horz; @border-radius)

Set sizing metrics for a button.

Parameters

Name Description Type
height Total height of the button. number
font-size Font size. number
line-height Line height. number
padding-horz Optional Left and right padding. number
border-radius Optional Border radius size. number

Usage

.btn-example {
    .button-size(30px; 12px; 1.5);
}


// Will output...
.btn-example {
    height: 30px;
    padding-top: 6px;
    padding-bottom: 6px;
    font-size: 12px;
    line-height: 1.5;
}
