gradient.directional mixin

#gradient.directional(@start-color; @end-color; @angle)

Apply a directional gradient to the background-image property of an element.

Parameters

Name Description Type
start-color Top color (start) for the gradient image. color
end-color Bottom color (end) for the gradient image. color
angle Optional The angle of the direction, specified in degrees. number

Usage

.gradient-example {
    #gradient.directional(#08c; darken(#08c, 15%));
}


// Will output...
.gradient-example {
    background-image: -webkit-linear-gradient(45deg, #08c, #005580);
    background-image: -o-linear-gradient(45deg, #08c, #005580);
    background-image: linear-gradient(45deg, #08c, #005580);
    background-repeat: repeat-x;
}
