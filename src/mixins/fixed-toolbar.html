fixed-toolbar mixin

.fixed-toolbar(@adminbar-top; @userbar-top)

Positions an element to be fixed to the top of the viewport.

Be sure to set a value for the @page-top-buffer variable when using this mixin.

JavaScript Dependency

This mixin relies on additional logic to handle when a user is logged in as either an admin or regular user. To enable this behavior, include this script at the bottom of your document before the closing </body> tag.


<script src="public/scripts/less/less.common.min.js"></script>

Parameters

Name Description Type
adminbar-top Expected height of the admin bar. number
userbar-top Expected height of the user bar. number

Usage

.page-header {
    .fixed-toolbar();
    
    height: 60px;
}


// Will output...
.page-header {
    width: 100%;
    top: 0;
    left: 0;
    position: fixed;
    
    height: 60px;
}
.is-admin .page-header {
    top: 60px;
}
.is-user .page-header {
    top: 40px;
}
