javelin-custom-bullets mixin

.javelin-custom-bullets(@bg-path; @position; @indent)

Apply a background image for custom list bullets in various Javelin modules.

Parameters

Name Description Type
bg-path The file path for the image to use as list bullets. string
position Optional The background image position. x y
indent Optional The starting position of the text content relative to the left edge of the list item. number

Usage

.javelin-custom-bullets("@{images-path}bullet-default.png"; 0 12px; 20px);
