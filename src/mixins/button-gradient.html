button-gradient mixin

.button-gradient(@start; @stop; @hover-start; @hover-stop)

Applies a gradient background image to the static, :hover, :focus and :active states of buttons.

Optional Arguments

If hover state colors are omitted in the arguments list, those values are still set as darkened versions of @start and @stop.

Parameters

Name Description Type
start Gradient start color (top). color
stop Gradient stop color (bottom). color
hover-start Optional Gradient start color for hover states. color
hover-stop Optional Gradient stop color for hover states. color

Usage

.btn-example {
    .button-gradient(#08c; darken(#08c, 15%); #f66; darken(#f66, 15%));
}


// Will output...
.btn-example {
    background-image: -webkit-linear-gradient(top, #08c 0%, #005580 100%);
    background-image: -o-linear-gradient(top, #08c 0%, #005580 100%);
    background-image: linear-gradient(to bottom, #08c 0%, #005580 100%);
    background-repeat: repeat-x;
    filter: progid:DXImageTransform.Microsoft.gradient(startColorstr='#ff0088cc', endColorstr='#ff005580', GradientType=0);
}

.btn-example:hover,
.btn-example:focus,
.btn-example:active,
.btn-example.active {
    background-image: -webkit-linear-gradient(top, #f66 0%, #ff1919 100%);
    background-image: -o-linear-gradient(top, #f66 0%, #ff1919 100%);
    background-image: linear-gradient(to bottom, #f66 0%, #ff1919 100%);
    background-repeat: repeat-x;
    filter: progid:DXImageTransform.Microsoft.gradient(startColorstr='#ffff6666', endColorstr='#ffff1919', GradientType=0); 
}
