make-grid-column-default mixin

.make-grid-column-default(@gutter)

Apply default grid column properties.

When using this mixin, you still need to define a width and a float for the column.

Parameters

Name Description Type
gutter Optional Gutter width. number

Usage

.example-column {
    .make-grid-column-default(30px);
    
    // You must still set a width and a float
    width: 20%;
    float: left;
}


// Will output...
.example-column {
    min-height: 1px;
    padding-left: 15px;
    padding-right: 15px;
    position: relative;
    
    width: 20%;
    float: left;
    
    -webkit-box-sizing: border-box;
       -moz-box-sizing: border-box;
            box-sizing: border-box;
}
