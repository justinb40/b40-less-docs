javelin-unfloat-images mixin

.javelin-unfloat-images(@selector; @remove-margins)

When dealing with shared module content between desktop and mobile websites, this function can be used to unfloat all floated images in Javelin modules.

Selector stacks will not work with the @selector argument. Only single selectors will correctly accept the styling from the mixin.

Parameters

Name Description Type
selector A single class name selector (without the decimal) to apply the mixin rules. keyword
remove-margins Optional Determines whether or not image margins should be set to zero. boolean

Usage

.javelin-unfloat-images(javelin_textItem);


// Will output...
.javelin_textItem img[style*="float"] {
    margin: 0 !important;
    float: none !important;
}


// This type of thing...
.javelin-unfloat-images(~".javelin_textItem, .javelin_galleryItem, .javelin_faqItem");

// Fails in LESS...
// (an additional decimal is created and the img[] qualifier does not apply to every selector in the stack)
..javelin_textItem,
.javelin_galleryItem,
.javelin_faqItem img[style*="float"] {
    margin: 0 !important;
    float: none !important;
}
