list-inline mixin

.list-inline(@offset-horz)

Easily put list items into one line and optionally add spacing between each item.

Parameters

Name Description Type
offset-horz Optional Left and right margins. number

Usage

.example-list {
    .list-inline(5px);
}


// Will output...
.example-list li {
    margin-left: 5px;
    margin-right: 5px;
    vertical-align: middle;
    list-style: none;
    display: inline-block;
}
.example-list li:first-child {
    margin-left: 0;
}
.example-list li:last-child {
    margin-right: 0;
}
