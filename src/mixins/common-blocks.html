common-blocks mixin

.common-blocks(@rules)

Apply a set of rules to the following selector: p, blockquote, ol, ul, table.

Parameters

Name Description Type
rules The styling rules to apply to the common blocks selector. ruleset

Usage

.index-section-1 {
    .common-blocks({
        &:last-child {
            margin-bottom: 0;
        }
    });
}


// Will output...
.index-section-1 p:last-child,
.index-section-1 blockquote:last-child,
.index-section-1 ol:last-child,
.index-section-1 ul:last-child,
.index-section-1 table:last-child {
    margin-bottom: 0;
}
