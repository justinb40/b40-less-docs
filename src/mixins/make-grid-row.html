make-grid-row mixin

.make-grid-row(@gutter)

Apply grid row properties to clear child floats.

Parameters

Name Description Type
gutter Optional Gutter width. number

Usage

.example-row {
    .make-grid-row(30px);
}


// Will output...
.example-row::before,
.example-row::after {
    content: "";
    display: table;
}
.example-row::after {
    clear: both;
}
.example-row {
    margin-left: -15px;
    margin-right: -15px;
}
