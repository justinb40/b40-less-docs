custom-bolditalic mixin

.custom-bolditalic(@bolditalic-family)

Set correct font styles and font families for custom fonts inside <b>, <strong>, <i> and <em> tags.

Parameters

Name Description Type
bold-family Font family to use for bold formatting. string
italic-family Font family to use for italic formatting. string
bolditalic-family Font family to use for bold italic formatting. string

Usage

.custom-bolditalic(@font-family-proximanova-bold; @font-family-proximanova-italic; @font-family-proximanova-bolditalic);


// Will output...
b,
strong {
    font-family: ProximaNova-Bold, Arial, Helvetica, sans-serif;
    font-weight: normal;
}

i,
em {
    font-family: ProximaNova-Italic, Arial, Helvetica, sans-serif;
    font-style: normal;
}

b em,
b i,
em b,
em strong,
i b,
i strong,
strong em,
strong i {
    font-family: ProximaNova-BoldItalic, Arial, Helvetica, sans-serif;
    font-style: normal;
    font-weight: normal;
}
