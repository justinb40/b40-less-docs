headings mixin

.headings(@rules)

Apply a set of rules to all six heading tags (h1 — h6).

Parameters

Name Description Type
rules The styling rules to apply to all six headings. ruleset

Usage

.index-section-1 {
    .headings({
        color: #fff;
    });
}


// Will output...
.index-section-1 h1,
.index-section-1 h2,
.index-section-1 h3,
.index-section-1 h4,
.index-section-1 h5,
.index-section-1 h6 {
    color: #fff;
}
