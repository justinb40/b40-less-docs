Scripts

Some of the components found in the LESS framework only come to life through JavaScript. There are also a few other scripts that don't pertain to built-in components, but still deliver additional functionality that is customizable through HTML5 data- attributes.

Location on filesystem

All minified scripts for the LESS framework are stored in the /public/scripts/less/ directory.

Including Scripts

Scripts should be included at the end of the DOM, just before the closing </body> tag. Including them at the end of the DOM guarantees that element selectors will locate matching elements without issues due to the fact that the entire DOM has been loaded. This does not include dynamically loaded elements.

Common Functions

All sites using the LESS framework should include include the necessary "common" script for the framework. This script acts as the main script for any site using the LESS framework and many of the built-in components and styles depend on it.

See full details of the less.common.js API
