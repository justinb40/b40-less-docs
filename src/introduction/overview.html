Back40 LESS Documentation

Introduction

This framework was created to enforce consistency and best practices in the Back40 frontend development process as well as expedite project turn-around. Much of the boilerplate code that we normally deal with has been hidden away, the built-in Javelin modules have been streamlined for development and there are several extra surprises along the way.

Overview

A prerequisite knowledge of CSS and LESS is required when using this framework.

LESS is an extension of CSS. Like other CSS pre-processors, LESS supports a CSS environment that is highly maintainable and conducive to a rapid development process by introducing variables and reusable rulesets.

When working on a new project, think about what you are building and try to relate your intentions to variables and tools found in the framework to simplify your code and your development process. A consistent and proper usage of these tools can produce very satisfying results!
