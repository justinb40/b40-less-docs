Compiler

In your project's compiler.less file, you will import the resources you wish to use in your project. You could choose to import the entire framework but this is likely a misuse because you may be importing more than you need which can produce significant code bloat.

Typical Imports

  • init.less imports core variables and mixins.
  • init-core.less imports core styles that you will always want.
  • init-javelin.less conveniently imports all Javelin resources.
  • init-utilities.less conveniently imports common utility helpers.
  • config.less imports your project settings and configuration.
  • theme.less imports your project theme styles.

Additional Imports

  • init-all.less conveniently imports all resources from the framework (use only for testing).

Including Custom Fonts From the Server

Unless you're working with a seamless font service like Google Fonts, you should import fonts from the public fonts repository on our server. If the font you need does not exist on the server, then we need to add it.

See the Using Custom Fonts section of this doc for more information.
