btn helper

This base class for buttons establishes button metrics and behaviors, but does not handle any extra style.

Usage

<!-- Button (typical usage with this helper) -->
<button type="button" class="btn btn-default">Button</button>


<!-- Hyperlinks (also typical usage with this helper) -->
<a href="#" class="btn btn-default">Button</a>


<!-- Inputs -->
<input type="button" value="Button" class="btn btn-default">
<input type="submit" value="Submit" class="btn btn-default">
