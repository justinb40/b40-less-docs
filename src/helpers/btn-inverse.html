btn-inverse helper

This button variant defines a button style that is the opposite of the .btn-default class. Colors are determined automatically using the spin() function in LESS, but this is just a logical placeholder that is meant to be overridden.

Gray values don't have a complimentary color. (i.e. they won't spin() to a different color)

Usage

<button type="button" class="btn btn-inverse">Inverse Button</button>
