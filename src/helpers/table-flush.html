table-flush helper

Remove left padding from the first cells in each row and remove right padding from the last cells in each row to allow table contents to touch the edges of the table.

Usage

<table class="table table-flush">
    <thead>
        <tr>
            <th>Table Cell</th>
            <th>Table Cell</th>
            <th>Table Cell</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Table Cell</td>
            <td>Table Cell</td>
            <td>Table Cell</td>
        </tr>
        <tr>
            <td>Table Cell</td>
            <td>Table Cell</td>
            <td>Table Cell</td>
        </tr>
        <tr>
            <td>Table Cell</td>
            <td>Table Cell</td>
            <td>Table Cell</td>
        </tr>
    </tbody>
<table>
