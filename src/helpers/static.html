static helper

Apply a static position to an element. This is helpful when you need to free up positioned containers so absolute positioned child elements can bind to different container.

Usage

<div class="row relative">
    <div class="grid-6 static">
        <div class="example-1">Some absolute positioned element that is now bound to the row element</div>
    </div>
    <div class="grid-6">
        <div class="example-2">Some absolute positioned element that is now bound to the grid-6 element</div>
    </div>
</div>
