table-condensed helper

Decrease table cell padding by about half (unless overridden with core variables).

Usage

<table class="table table-condensed">
    <thead>
        <tr>
            <th>Table Cell</th>
            <th>Table Cell</th>
            <th>Table Cell</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Table Cell</td>
            <td>Table Cell</td>
            <td>Table Cell</td>
        </tr>
        <tr>
            <td>Table Cell</td>
            <td>Table Cell</td>
            <td>Table Cell</td>
        </tr>
        <tr>
            <td>Table Cell</td>
            <td>Table Cell</td>
            <td>Table Cell</td>
        </tr>
    </tbody>
<table>
