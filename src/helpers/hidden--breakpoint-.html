hidden-[ breakpoint ] helper

Make an element hidden within a specific media query breakpoint.

Usage

<p class="hidden-xs">Extra small screen / phone</p>
<p class="hidden-sm">Small screen / phone</p>
<p class="hidden-md">Medium screen / tablet</p>
<p class="hidden-lg">Large screen / desktop</p>
<p class="hidden-xl">Extra large screen / wide desktop</p>
