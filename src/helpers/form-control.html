form-control helper

Apply beautifying, cross-browser styles to form controls including typography, spacing, backgrounds, borders, placeholder text and focus ring color.

Styling Limitations

Select menus are limited by how much styling we can apply to them. The dropdown caret (arrow) is untouchable and styling <option> tags does not often carry across all browsers and devices. On top of that, some versions of Webkit browsers show a grayish gradient as the background of the select menu, which is not entirely pretty.

Usage

<!-- Text Input -->
<input type="text" class="form-control" placeholder="Text Input">


<!-- Textarea -->
<textarea class="form-control" placeholder="Textarea"></textarea>


<!-- Select Menu -->
<select class="form-control">
    <option value="1">Option 1</option>
    <option value="2">Option 2</option>
    <option value="3">Option 3</option>
</select>
