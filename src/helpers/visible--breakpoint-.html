visible-[ breakpoint ] helper

Make an element visible within a specific media query breakpoint.

Usage

<p class="visible-xs">Extra small screen / phone</p>
<p class="visible-sm">Small screen / phone</p>
<p class="visible-md">Medium screen / tablet</p>
<p class="visible-lg">Large screen / desktop</p>
<p class="visible-xl">Extra large screen / wide desktop</p>

<p class="visible-nav-down">Below the responsive navigation breakpoint</p>
<p class="visible-nav-up">Above the responsive navigation breakpoint</p>
