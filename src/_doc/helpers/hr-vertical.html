hr-vertical helper

Useful for vertical pin lines between sections of content. By default, the vertical line is positioned to the center of its nearest positioned ancestor. Override this position with .offset-[n] to position the vertical line between two grid columns. If this is not sufficient for positioning, explicitly set the margin-left property in a [style] attribute.

Helpful when used in conjunction with .relative to specify the nearest positioned ancestor.

Usage

<div class="row relative">
    <div class="grid-3">
        <p>Column 1</p>
    </div>
    <hr class="hr-vertical offset-3">
    <div class="grid-3">
        <p>Column 2</p>
    </div>
    <hr class="hr-vertical offset-6">
    <div class="grid-3">
        <p>Column 3</p>
    </div>
    <hr class="hr-vertical offset-9">
    <div class="grid-3">
        <p>Column 4</p>
    </div>
</div>
