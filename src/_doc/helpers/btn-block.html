btn-block helper

Use this button variant to make a button span the width of its container.

Multiple consecutive .btn-block elements will add a small amount of space above each consecutive button.

Usage

<button type="button" class="btn btn-default btn-block">Block Button</button>
<button type="button" class="btn btn-default btn-block">Block Button</button>
<button type="button" class="btn btn-default btn-block">Block Button</button>
