Icon List component

Use icon lists to easily manage a list of icons which may or may not be accompanied by a text label. By default, hovering over each item will produce a 50% opacity hover state to avoid dealing with additional images for icon hover states. Control spacing between list items with the @icon-list-offset-horz variable.

Classes

Class Usage
icon-list Container for an icon list element.
