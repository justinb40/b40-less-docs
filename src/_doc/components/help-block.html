Help Block component

Provide additional information about a form field with help blocks.

Classes

Class Usage
help-block Wrap text in this class to create a minimized line of text to offer additional information about a form field.

Usage

<div class="form-group">
    <label>Email Address</label>
    <input type="text" class="form-control">
    <span class="help-block">We will not share your email address with third parties.</span>
</div>
