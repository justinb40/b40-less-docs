Button Group component

Merge buttons together into a single component.

Classes

Class Usage
btn-group Wrap .btn elements in this class to create a group of buttons joined together.

Usage

<div class="btn-group">
    <button type="button" class="btn btn-default">Button</button>
    <button type="button" class="btn btn-default">Button</button>
    <button type="button" class="btn btn-default">Button</button>
</div>
