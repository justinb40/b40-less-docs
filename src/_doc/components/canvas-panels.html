Canvas Panels component

Off-canvas navigation made easy. Side panels may be toggled by using either the .open-left-canvas or the .open-right-canvas class on a button control.

JavaScript Dependency

This component relies on additional logic to handle revealing and dismissing the canvas panels. Include these scripts at the bottom of your document before the closing </body> tag.

Optionally, include hammer.min.js to allow the user to dismiss a side panel by swiping.


<script src="public/scripts/hammer.min.js"></script>
<script src="public/scripts/less/less.common.min.js"></script>
<script src="public/scripts/less/less.canvas-panels.min.js"></script>

Classes

Class Usage
canvas Container for the main panel which should contain the content of the page. This element will shift to the left or right when it is the descendant of either the canvas-left-is-open or canvas-right-is-open class.
canvas-left Container for the left canvas panel.
canvas-right Container for the right canvas panel.

JavaScript Classes

Class Usage
open-left-canvas Use this class on a clickable element to toggle the canvas-left-is-open class on the <body>, which toggles the visibility of the left canvas panel.
open-right-canvas Use this class on a clickable element to toggle the canvas-right-is-open class on the <body>, which toggles the visibility of the right canvas panel.

Usage

<div class="page">
    <div class="canvas">
        <header class="page-header">
            
            <h2 class="logo">
                <img src="/sites/sitename/templates/mobile/images/logo@2x.png" alt="Sitename">
            </h2>
            <button type="button" class="navicon open-right-canvas">
                <span class="sr">Toggle Navigation</span>
                <span class="navicon-bar"></span>
                <span class="navicon-bar"></span>
                <span class="navicon-bar"></span>
            </button>
            
        </header>
        
        <main class="page-main">
            ...
        </main>
        
        <footer class="page-footer">
            ...
        </footer>
    </div>
    <div class="canvas-right">
        <!nav id="nav" type="nested" /!>
    </div>
</div>

JavaScript Events

Event Usage
b40.canvas.left.open Broadcast when the left canvas panel is opened.
b40.canvas.right.open Broadcast when the right canvas panel is opened.
b40.canvas.left.closed Broadcast when the left canvas panel is closed.
b40.canvas.right.closed Broadcast when the right canvas panel is closed.

Using Events

// Bind to the `document` to listen for canvas panel events
$('body').bind('b40.canvas.right.open', function() {
    // code goes here
});
