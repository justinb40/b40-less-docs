Alert component

Provide contextual feedback messages for typical user actions with alert messages.

Classes

Class Usage
alert Wrap text in this base class paired with a contextual class to create an alert box.
alert-link Apply to hyperlinks within the alert box to handle color variations.
alert-title Apply to <h4> elements to handle color and spacing variations.
alert-info Contextual class for conveying an informational message.
alert-success Contextual class for conveying a successful message.
alert-warning Contextual class for conveying a warning message.
alert-danger Contextual class for conveying an error message.

Usage

<div class="alert alert-success">
    <h4 class="alert-title">Alert Title</h4>
    <p>Lorem ipsum dolor sit amet, <a href="#" class="alert-link">consectetur adipisicing</a> elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.</p>
</div>
