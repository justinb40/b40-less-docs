Weather module

#javelin-weather

The core Weather module uses the Yahoo Weather API to display current weather information based on the zip-code provided in the module settings.

Weather Icons

The icons used in this module are not customizable because the data that determines which icon to display comes from an API service and each icon is an <img> tag, which gives us very little to work with.

Mixins for Legacy Selectors

Source: /less/javelin/weather.less

Mixin Usage
current Current conditions block in the weather display.
forecast Forecast block in the weather display.
icon Icon block in the weather display.
title Title block in the weather display.
temp Current temperature block in the weather display.
time Current date and time block in the weather display.
