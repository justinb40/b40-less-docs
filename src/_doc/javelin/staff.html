Staff module

#javelin-staff

The core Staff module in Javelin is possibly the most difficult module to work with when it comes to custom layouts or other styling. Several LESS mixins for this module make it easier to style specific elements found within a staff listing including .

Thumbnail Sizing

The size of the thumbnail generated through module settings that are editable by the user should be handled with CSS as well as module settings. To set this size globally, override the @staff-item-thumb-width var. To set this value on individual staff modules, you have to manually provide the CSS to handle the sizing.
