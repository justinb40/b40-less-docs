list-unstyled mixin

.list-unstyled(@deep)

Remove bullets from list items. Additionally, you can specify whether immediate children or all descendants are affected.

Parameters

Name Description Type
deep Optional Determines whether immediate children or all descendants are affected. boolean

Usage

.example-list-1 {
    .list-unstyled();
}

.example-list-2 {
    .list-unstyled(true);
}


// When @deep is set to FALSE, only immediate child list items are affected
.example-list-1 {
    margin-left: 0;
}
.example-list-1 > li {
    list-style: none;
}


// When @deep is set to TRUE, all descendant list items are affected
.example-list-2 {
    margin-left: 0;
}
.example-list-2 li {
    list-style: none;
}
