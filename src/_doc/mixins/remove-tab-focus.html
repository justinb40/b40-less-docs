remove-tab-focus mixin

.remove-tab-focus()

This mixin will nuke the focus outline in WebKit browsers.

You must apply this mixin to the normal selector. Not the :focus state.

Parameters

None.

Usage

.example-input {
    .remove-tab-focus();
}


// Will output...
.example-input {
    -webkit-tap-highlight-color: rgba(0, 0, 0, 0) !important; 
    -webkit-focus-ring-color: rgba(0, 0, 0, 0) !important; 
    
    outline: none !important;
}
