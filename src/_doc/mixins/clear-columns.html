clear-columns mixin

.clear-columns(@num)

Control where column overflow clears new rows.

Some rows are meant to be overflowed. When a row has an overflow of columns, it attempts to push the columns that won't fit in the remaining space to the next line.

However, varying sizes of content per column may leave us without the neat structure of rows and columns that we want. This mixin will help us enforce when to start and stop a new row when we are dealing with rows that have an overflow of columns.

Parameters

Name Description Type
num Total number of columns per row. number

Usage

.parent {
    .grid-3 {
        .clear-columns(4);
    }
}


// Will output...
.parent .grid-3:nth-child(4n+1) {
    clear: left;
}

.parent .grid-3:nth-child(4n+4) {
    clear: right;
}
