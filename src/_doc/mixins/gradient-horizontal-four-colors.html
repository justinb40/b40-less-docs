gradient.horizontal-four-colors mixin

#gradient.horizontal-four-colors(@start-color; @inner1-color; @inner2-color; @end-color; @inner1-percent; @inner2-percent)

Apply a horizontal, four-color gradient to the background-image property of an element.

Parameters

Name Description Type
start-color Left color (start) for the gradient image. color
inner1-color First inner color for the gradient image. color
inner2-color Second inner color for the gradient image. color
end-color Right color (end) for the gradient image. color
inner1-percent Optional Color-stop position for the first inner color. percent
inner2-percent Optional Color-stop position for the second inner color. percent

Usage

.gradient-example {
    #gradient.horizontal-four-colors(#000; fade(#000, 0%); fade(#000, 0%); #000; 30%; 70%);
}


// Will output...
.gradient-example {
    background-image: -webkit-linear-gradient(top, #000000, rgba(0, 0, 0, 0) 30%, rgba(0, 0, 0, 0) 70%, #000000);
    background-image: -o-linear-gradient(top, #000000, rgba(0, 0, 0, 0) 30%, rgba(0, 0, 0, 0) 70%, #000000);
    background-image: linear-gradient(to bottom, #000000, rgba(0, 0, 0, 0) 30%, rgba(0, 0, 0, 0) 70%, #000000);
    background-repeat: repeat-y;
    filter: progid:DXImageTransform.Microsoft.gradient(startColorstr='#ff000000', endColorstr='#00000000', GradientType=0);
}
