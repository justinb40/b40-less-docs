visibility mixin

.visibility()

Set an element to display as block.

This is typically only used within the definitions of the responsive visibility classes.

Parameters

None.

Usage

.example {
    .visibility();
}


// Will output...
.example {
    display: block !important;
}
table.example {
    display: table !important;
}
tr.example {
    display: table-row !important;
}
th.example,
td.example {
    display: table-cell !important;
}
