bg-cover mixin

.bg-cover(@position)

Apply "cover" style background sizing properties to a ruleset.

Parameters

Name Description Type
position The value for the background-position property. x y

Usage

.index-section-1 {
    .bg-cover();
}

.index-section-2 {
    .bg-cover(center);
}


// Will output...
.index-section-1 {
    background-position: center top;
    background-repeat: no-repeat;
    background-size: cover;
}

.index-section-2 {
    background-position: center;
    background-repeat: no-repeat;
    background-size: cover;
}
