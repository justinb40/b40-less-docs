progress-bar-size mixin

.progress-bar-size(@height; @border-radius)

Set sizing metrics for a progress bar.

Parameters

Name Description Type
height Total height of the progress bar. number
border-radius Border radius. number

Usage

.example-progress-bar {
    .progress-bar-size(30px; 8px);
}


// Will output...
.example-progress-bar {
    height: 30px;
    border-radius: 8px;
}
