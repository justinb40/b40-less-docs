javelin-custom-images mixin

.javelin-custom-images(@rules; @anti-rules)

Define custom image styling rules for images in various Javelin modules. If @anti-rules is omitted, the .img-nostyle rules will not be included.

Parameters

Name Description Type
rules The styling rules to apply to images. ruleset
anti-rules Optional The styling rules to reset @rules. ruleset

Usage

.javelin-custom-images({
    border: 2px solid @border-color-base;
    border-radius: @border-radius-base;
}; {
    border: 0;
    border-radius: 0;
});
