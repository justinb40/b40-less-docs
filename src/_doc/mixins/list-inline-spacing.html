list-inline-spacing mixin

.list-inline-spacing(@offset-horz)

Add spacing between each <li> in a list.

Best used in conjunction with the .list-inline() mixin.

Parameters

Name Description Type
offset-horz Left and right margins. number

Usage

.example-list {
    .list-inline-spacing(5px);
}


// Will output...
.example-list li {
    margin-left: 5px;
    margin-right: 5px;
}
.example-list li:first-child {
    margin-left: 0;
}
.example-list li:last-child {
    margin-right: 0;
}
